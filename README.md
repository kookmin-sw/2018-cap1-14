### 1. 프로잭트 소개

YouTube의 URL에서 음원을 추출하여 악보를 생성하는 웹 어플리케이션 개발을 목표로 한다.
MusicSeat에서는 다음과 같은 기능들을 구현하고자 한다.
- YouTube의 음원을 추출하여 악보를 생성
- 악보를 재생할 수 있는 기능
- 악기연주를 피아노, 기타, 바이올린 등 으로 선택할 수 있는 기능
- 악보 음정에 맞춰 키를 누르는 게임 기능
